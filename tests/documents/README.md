# Test Documents for Unoserver

This folder contains test files used by the test suite.

## Required for Slideshow Tests

Please add the following file:

- `presentation_test.odp` (or `.pptx`)
  - A simple presentation with at least 8 slides
  - Can contain text, images, and basic transitions
  - Used by `test_slideshow.py`

---

**Current conversion test files:**

- `simple.odt`
- `simple.xlsx`
- `simple.encrypted.odt`
- `index-with-fields.odt`

You can copy any existing `.odp` or `.pptx` file here and rename it to `presentation_test.odp` for slideshow testing.
